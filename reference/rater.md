# Variable ratings for 1287 examinees, 7 variables and 4 raters.

This data set contains fictional ratings of achievement test items of
1287 examinees, 7 variables and 4 raters in the long format.

## Usage

``` r
data(rater)
```

## Format

'data.frame': 4200 obs. of 4 variables

- id:

  Student identifier

- rater:

  rater identifier

- variable:

  Variable identifier

- value:

  The rating given by the corresponding rater to the response of the
  student to the variable

## Source

Simulated data
